# Agregar estas líneas a .gitignore

```
# Environment variables
.env

# Storage and work directories
storage/
.work/

# Repos configuration (keep example only)
repos.json

# Server build
server/dist/
```

**Nota**: Asegúrate de que `.env.example` y `repos.json.example` SÍ estén trackeados en git, pero los archivos sin `.example` NO.
